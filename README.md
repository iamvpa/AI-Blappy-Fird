# AI-Blappy-Fird
 A copy of the famous flappy bird game, only this time the computer plays instead of you!
## Requirements
* Python 3.6+
* NumPy
 ```pip install numpy```
* Pygame
 ```pip install pygame```
* MatplotLib
```pip install matplotlib```
* NEAT 
```pip install neat-python```
* Graphviz 
```pip install graphviz```
